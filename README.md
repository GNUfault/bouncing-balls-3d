## Prerequisites
1.) You will need to install:
```
git python3 python3-pip
```
2.) Then run:
```
git clone https://github.com/bluMATRIKZ/bouncing-balls-3d.git && \
cd bouncing-balls-3d && \
python3 -m pip install -r requirements.txt
```
4.) Run main.py:
```
python3 main.py
```
